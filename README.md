# reimagined-dream
smart_home_prototype
a mixture made of esp32's and a raspberry pi. 
utilizes:
* esp now
* uart
* web sockets
* sqlite3 database

# required:

* esp32 slave x 4 
* esp32 master x 1
* dht11 x 4 
* pir sensor x 4
* infrared proximity sensor 4
* raspberrypi (4 or 5) x 1
